## Getting Started

*<em> - Clone project to your local machine: </em>*

```bash
git clone https://github.com/gk-daniel/v-card.git
```  

*<em> - Navigate to the project directory using the cd command: </em>*

```bash
cd v-card
```  

*<em> - Open index file in browser  </em>*

```css
index.html
```
